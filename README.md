# Arena Preference Prototype

Small Unity prototype for testing action RPG combat preferences: skill rhythm, positioning, enemy pressure, loot/equipment changes, and repeatable wave rewards.

## Controls

- Move: WASD or Arrow Keys
- Aim: Mouse
- Basic skillshot: Left Mouse / 1
- Dash: Space / 2
- Area skill: Q / 3
- Defensive support buff: E / 4
- Toggle debug tuning panel: F1

## Loop

Press Start Wave, clear enemies, pick one reward, then start the next wave. Loot cubes can also drop from enemies and auto-equip on pickup.

The generated scene is `Assets/Scenes/ArenaPrototype.unity`.
